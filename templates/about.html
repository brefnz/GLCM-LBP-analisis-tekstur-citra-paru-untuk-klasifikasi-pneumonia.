LungClassify

LungClassify adalah aplikasi klasifikasi penyakit paru-paru berbasis pengolahan citra digital. Aplikasi menggunakan kombinasi metode ekstraksi fitur tekstur tradisional (GLCM, LBP) dengan fitur statistik, dan melatih tiga classifier secara otomatis untuk memilih model dengan performa terbaik.

Sistem dirancang untuk mengklasifikasikan citra rontgen dada menjadi dua kelas klinis: Normal dan Pneumonia. Gambar yang tidak dikenali sebagai citra rontgen ditangani secara otomatis menggunakan mekanisme confidence threshold tanpa memerlukan kelas dataset tambahan.

Dataset yang Direkomendasikan
Chest X-Ray Images (Pneumonia)
Kaggle — Paul Mooney
5.856 gambar total NORMAL: 1.583 PNEUMONIA: 4.273

Dataset berisi citra rontgen dada dari pasien anak-anak. Sistem secara otomatis melakukan undersampling untuk menyeimbangkan kedua kelas saat training.

Buka di Kaggle
COVID-19 Radiography Database
Kaggle — Tawsifur Rahman et al.
21.165 gambar NORMAL: 10.192 Lebih seimbang

Alternatif dataset yang lebih besar dan lebih seimbang. Gunakan folder NORMAL dan COVID sebagai dua kelas (rename folder PNEUMONIA sesuai kebutuhan).

Panduan Instalasi
  1. Install dependensi Python pip install -r requirements.txt
  2. Siapkan dataset
    Letakkan gambar ke dalam folder: dataset/ NORMAL/ (gambar paru-paru normal) PNEUMONIA/ (gambar paru-paru dengan pneumonia)
  3. Jalankan aplikasi python app.py
  4. Buka browser
    Akses aplikasi di http://localhost:5000
  5. Lakukan training
    Buka menu Training Model, lalu klik "Mulai Training". Setelah training selesai, model tersimpan otomatis dan tidak perlu dilatih ulang.
Metode yang Digunakan
1. Pra-pemrosesan Citra
  • Konversi ke grayscale
  • Resize ke 256x256 piksel
  • CLAHE (Contrast Limited Adaptive Histogram Equalization) untuk peningkatan kontras
2. Ekstraksi Fitur GLCM
  • Gray-Level Co-occurrence Matrix (GLCM)
  • 2 jarak (d=1, d=2) × 4 sudut (0°, 45°, 90°, 135°)
  • 6 properti: Contrast, Dissimilarity, Homogeneity, Energy, Correlation, ASM
  • Total: 60 fitur GLCM
3. Ekstraksi Fitur LBP
  • Local Binary Pattern — uniform method
  • Radius = 3, titik = 24
  • Histogram ternormalisasi: 26 bin
4. Fitur Statistik
  • Mean, Std, Skewness, Kurtosis
  • Persentil (P10, P25, P75, P90)
  • Energi intensitas, Variansi
  • Total: 10 fitur statistik
5. Klasifikasi
  • SVM dengan kernel RBF (C=10, gamma=scale)
  • Random Forest (200 estimator)
  • Gradient Boosting (150 estimator, lr=0.1)
  • Model terbaik dipilih via 5-Fold Stratified Cross-Validation
  • StandardScaler untuk normalisasi fitur
6. Deteksi Gambar Non-Paru (2 Layer)
  • Layer 1 — Isolation Forest: dilatih pada seluruh fitur rontgen paru-paru. Gambar di luar distribusi rontgen (foto benda, selfie, dll.) langsung ditolak sebagai outlier sebelum masuk ke classifier
  • Layer 2 — Confidence Threshold (55%): jika Isolation Forest meloloskan gambar namun classifier tidak cukup yakin, hasil tetap dikembalikan sebagai "Tidak Dikenali"
  • Tidak memerlukan dataset kelas ketiga
